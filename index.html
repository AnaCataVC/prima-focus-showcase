Una aplicación de gestión de tareas "local-first" construida nativamente para Android. Utiliza un avanzado sistema predictivo para seleccionar tu "Tarea de Hoy".

Descargar APK (Android) Ver Características

Diseñada para la Productividad

Tecnologías modernas y arquitectura sólida para garantizar el mejor rendimiento y privacidad.

Local-First (Privacidad)

Toda tu información se guarda de forma segura en tu dispositivo mediante base de datos Room. Sin nube, sin rastreo.

Puntuación Predictiva

Un algoritmo inteligente que analiza tus tareas y resalta la "Tarea de Hoy" para evitar la parálisis por análisis.

Nativa en Kotlin

Desarrollada utilizando Jetpack Compose y Kotlin para ofrecer una experiencia fluida, animaciones suaves y un diseño Material 3 impecable.

Notificaciones Inteligentes

WorkManager integrado para recordatorios sutiles y actualizaciones en segundo plano que te mantienen enfocado.

Modo Oscuro

Interfaz diseñada meticulosamente con soporte para dark mode, ideal para concentrarte de noche sin cansar la vista.

Fácil de Usar

Arquitectura enfocada en el usuario sin menús complejos. Entra, mira tu tarea principal y comienza a actuar.
